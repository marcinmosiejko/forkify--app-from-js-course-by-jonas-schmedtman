<!-- RUN LOCALLY -->
## Run The App Locally

1. Clone the repo
   ```sh
   git clone https://github.com/mosiej803/crypto-calc-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run local server
   ```sh
   npm start
   ```
